## Q1: 知乎提问：我该不该放手?

**场景**: 楼主刚毕业，与女友相识两个月后在一起。女友要求必须在一线城市有房才结婚，楼主是农村出身，5线小城市长大，月薪9k，女友月薪5k但是一线城市本地人。楼主拼命赚钱但压力巨大。

**涉及人物**:
- **楼主**: 男性，刚毕业，农村出身，月薪9k，拼命加班赚钱，承担巨大经济压力
- **女友**: 一线城市本地人，要求男方有房才结婚，月薪5k

**心理学维度**:
- 楼主: OCEAN ≈ (O:0.4, C:0.7, E:0.4, A:0.5, N:0.7) — 高尽责(拼命工作)、高神经质(压力巨大)。依恋: 偏焦虑型(害怕失去)。防御: 合理化("只能靠自己搞定")。ACE: 2-3(经济不安全感可能来自早期经历)。图式: 屈从(无条件满足对方要求)、缺陷/羞耻(农村出身 vs 一线城市)
- 女友: OCEAN ≈ (O:0.3, C:0.5, E:0.5, A:0.3, N:0.5) — 低宜人(物质要求刚性)。依恋: 安全型(自身条件有保障)
